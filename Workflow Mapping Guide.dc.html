Core Idea

The best AI improvements begin with the work, not the tool. Managers often ask, "Where can we use AI?" A better question is "What part of this work creates the most friction and the least value for people?" Start there.

Before You Start

The best first workflows are repeatable, predictable, important enough to matter, and small enough to improve within a few weeks.

Don't start with the biggest or most broken process. Early wins build confidence, demonstrate value, and teach the team how to improve the next workflow.

Chapter Three

Choose the Improvement

Manager Insight

Protect the Purpose of the Work

Before introducing AI into a workflow, ask:

What is the real purpose of this work?

Use AI to strengthen that purpose—not replace it.

If the highest value comes from preparing information, organizing data, or reducing repetitive effort, AI can often create significant value.

If the highest value comes from human judgment, discussion, coaching, relationship-building, or decision-making, AI should support those activities—not replace them.

A common question

"Can AI do this?"

A better question

"Should AI do this?"

Use AI to remove friction from the work—not the value from the work.
